# UnityStandardUtils


Unity基础工具库


---


提供了Unity开发的一些常用功能：

加密编解码

存取档

语言本地化及对应的可视化编辑器

单例MonoBehavior的实现

可以传参的SceneLoader

键位管理器

玩家背包模拟

简单POST，GET请求

SocketStuff引擎:（不再维护，更名并转移至[SGNF](https://github.com/RyuBAI/SGNF)项目下）


---


使用请打包dll，并和Newtonsoft.Json.dll放在项目Asset目录下

[具体使用方法请查看Wiki](https://github.com/RyuBAI/UnityStandardUtils/wiki)
